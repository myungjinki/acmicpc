# [크레인 인형뽑기 게임](https://programmers.co.kr/learn/courses/30/lessons/64061)

- 시작 시간: 10:30
- 종료 시간: 11:25
- 소요 시간: 55분

너무 쉬운데, 자바스크립트가 익숙하지 않아서 오래 걸렸다.

```
function solution(board, moves) {
  var arr = [];
  var answer = 0;
  moves.map((m, idx) => {
    for (let b of board) {
      if (b[m - 1] != 0) {
        arr.push(b[m - 1]);
        b[m - 1] = 0;
        break;
      }
    }
  });
  var arr2 = [];
  arr.map((r) => {
    if (arr2[arr2.length - 1] == r) {
      answer += 2;
      arr2.pop(r);
    } else {
      arr2.push(r);
    }
  });
  return answer;
}
```
