# 피드백

- sort를 사용한다는 생각을 너무 오래 못했다.

```
function solution(participant, completion) {
  participant.sort();
  completion.sort();
  var i;
  for (i = 0; i < participant.length; i++) {
    if (participant[i] != completion[i]) {
      break;
    }
  }

  return participant[i];
}

```
