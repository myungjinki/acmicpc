### 알고리즘 학습방법 - 문제 쪼개기

알고리즘 문제를 풀 때의 단계를 생각해볼까요?

1. 문제를 읽고 파악한다.
2. 파악한 후 어떤 자료구조를 사용할 지 정한다. (자료구조를 모르면 주먹구구)
3. 어떤 알고리즘을 사용할 것인지 정한다. (알고리즘을 모르면 주먹구구)
4. 예외처리와 반례를 생각한다. (생각하지 않으면 예외가 발생한다)
5. 선택한 언어로 코드를 구현한다.
